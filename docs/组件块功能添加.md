# 组件块功能拓展

## icon组件

现在缺乏这个组件，lucide-react icon可以提供一些电商网站常用的icon给到运营人员快速选择，然后更多的icon就让他们跳转到https://lucide.dev/icons/ 去选择icon 复制icon的组件名，黏贴到这个弹窗的输入框里，点击done后能在编辑器里预览图标就最好了。

## 链接跳转组件

ui结构如下

|       | 标题文本                       |
| icon  | ----------------------------- |
|       | 链接跳转（展示文本 + 实际跳转url） |

左侧的icon 需要做两种方式，一种是用lucide-react 组件的名字，另一种则是运营人员从media里选择图片。提供三个文本输入框，标题文本（可不填），展示文本，跳转url（这个我们稍后探讨如何让运营人员可视化选择站内的快速链接）。



## 循环播放走马灯链接条

* 自动循环轮播，轮播方向为item不断向左移动（当播放至最后一个item时，自动在右侧继续连接最开始的item）
* 每个item包含类似上面的链接跳转组件。
* 运营人员可以在这个组件里添加 删除 拖拽排序这些item。
* 可控制滚动速度。



## 轮播组件拓展

目前的轮播组件的item里只有title，media和description，需要再增加多一个button，button默认渲染在description下方左侧对齐，需要提供这些选项，是否启用button，启用后需输入button的文本，跳转链接。
